# Репозиторий по проекту Телеграмм бот для анализа отзывов.

## Description
Создать прототип Telegram-бота, способного принимать голосовые и текстовые сообщения от пользователей, автоматически анализировать их тональность, определять тему обращения, выявлять наличие просьбы о помощи и предоставлять краткое содержание (саммари) сообщения.
